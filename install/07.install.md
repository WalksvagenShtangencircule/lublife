## Install app

### Рекомендуемый способ: единый deploy-скрипт

Скрипт `install/scripts/rbt-deploy.sh` поддерживает два режима:
- `install` — чистая установка;
- `update` — обновление существующей инсталляции.

#### 1) Чистая установка SmartAccess (форк)

```bash
sudo ./install/scripts/rbt-deploy.sh install \
  --repo git@github.com:WalksvagenShtangencircule/SmartAccess.git \
  --ref main \
  --target /opt/rbt
```

Если нужно сразу инициализировать БД:

```bash
sudo ./install/scripts/rbt-deploy.sh install \
  --repo git@github.com:WalksvagenShtangencircule/SmartAccess.git \
  --ref main \
  --target /opt/rbt \
  --init-db \
  --admin-password 'CHANGE_ME_STRONG_PASSWORD'
```

#### 2) Чистая установка upstream SmartYard

```bash
sudo ./install/scripts/rbt-deploy.sh install \
  --repo git@github.com:rosteleset/SmartYard-Server.git \
  --ref main \
  --target /opt/rbt
```

#### 3) Что делает скрипт

- клонирует/обновляет git-репозиторий;
- ставит зависимости `composer` в `server`;
- подтягивает клиентские библиотеки в `client/lib`;
- собирает `Leaflet`;
- создает `config.json` из sample-файлов, если их еще нет;
- выполняет `server/cli.php --strip-config` для серверного конфига;
- по флагу `--init-db` запускает `init-db`, `init-clickhouse-db`, `admin-password`, `reindex`, `install-crontabs`.

### Ручная установка (если нужен полный контроль)

```bash
git clone git@github.com:WalksvagenShtangencircule/SmartAccess.git /opt/rbt
cd /opt/rbt
git checkout main
cd /opt/rbt/server && composer install
cd /opt/rbt/client/lib
git clone --branch v3.2.0 https://github.com/ColorlibHQ/AdminLTE
git clone https://github.com/davidshimjs/qrcodejs
git clone https://github.com/ajaxorg/ace-builds/
git clone --branch v1.9.2 https://github.com/Leaflet/Leaflet
cd Leaflet && npm install && npm run build
cp /opt/rbt/client/config/config.sample.json5 /opt/rbt/client/config/config.json
cp /opt/rbt/server/config/config.sample.json5 /opt/rbt/server/config/config.json
php /opt/rbt/server/cli.php --strip-config
```

После этого вручную заполните конфиги:
- `/opt/rbt/server/config/config.json`
- `/opt/rbt/client/config/config.json`
